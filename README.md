# Élégance Parfum

Luxury Perfume & Beauty Ecommerce Frontend

## Tech Stack

- Next.js
- React
- Tailwind CSS

## Features

- Feminine, luxury-inspired UI (Chanel Beauty, Sephora)
- Responsive, mobile-first design
- Sticky navbar, hero section, product grid, cart UI
- Beautiful typography and color palette

## Getting Started

```bash
npm install
npm run dev
```

## Folder Structure

- `/app` — Next.js app directory
- `/components` — UI components
